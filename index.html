Projects
3D Renderer with Entity-Component-System

This was my Bachlorthesis project.
A full 3D Renderer with forward shading in DirectX 12.
To update all Entities I've implemented a custom Entity-Component-System.

Most interesting features:

  • Frustum culling
  • Transparancy
  • Shadow map
  • Skybox
  • PhysX integration

Frameworks/Libraries used: DX12, Win32, Imgui and PhysX.

Co-op teraforming game

This is a university project with 3 other Students. My responsibility was to implement the brushes for the manipulation of the terrain and Managing the Team.
I used compute shaders for this task to manipulate the Heightmap of the terrain.
Also, I implemented the shader for the blending of the texture to make the terrain change these on defined heights.
Helping out my teammates was an important part too.

Frameworks/Libraries used: Unity.

Couch co-op jump and run game

This Project is for a four-day game jam at my university I made with a teammate.
Two players are connected by a rope and need to support each other to climb the tower.
An important part of the gameplay is the behavior of the rope and how the players can manipulate the length of it.


Frameworks/Libraries used: Unity.
